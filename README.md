## For start this server in localhost you need to do those instruction

### clone this project

```shall
    https://github.com/saiyedimtiaj/simple-mongoose.git
```

### change directory

```shall
    cd simple-mongoose
```

### install node modules

```shall
    npm i
```

### Now create env file and see .test.env file

### start the server

```shall
    npm run start:dev
```
